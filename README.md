# Backend (Node.js Learning Project)

This is a simple backend project built while learning Node.js and Express.js.  
It helps understand how APIs, routing, and server-side logic work.

---

## 🚀 What I Learned

- How to create a Node.js server
- Basics of Express.js
- Creating REST APIs
- Handling routes and requests
- Using middleware
- Connecting to a database 

---

## 🛠️ Tech Stack

- Node.js
- Express.js
-  MongoDB / MySQL
- Postman (for testing APIs)
