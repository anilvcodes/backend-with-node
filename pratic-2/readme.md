# 🛡️ Zod + Express

A simple backend API built using **Node.js**, **Express**, and **Zod** for strong input validation.

---

## 🚀 Features

* User input validation using Zod
* Health check validation (array of numbers)
* Clean error handling
* Simple REST API structure
* Beginner-friendly backend project

---

## 🛠️ Tech Stack

* Node.js
* Express.js
* Zod (Schema validation)
