# 🧠 Health Check Backend (Express.js)

A simple backend server built using **Express.js** that demonstrates:

* Middleware usage
* Handling GET & POST routes
* Basic request processing
* Global error handling

---

## 🚀 Features

* ✅ JSON parsing using middleware
* ✅ GET route for testing server
* ✅ POST route for handling health data
* ✅ Global error handler (catch all errors)

---

## 📦 Tech Stack

* Node.js
* Express.js
